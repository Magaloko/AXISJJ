# Mitglied — Bedienungsanleitung

Alles was du als Mitglied in der AXIS JJ App tun kannst.

## Navigation

Nach dem Login erscheinst du auf deinem **Dashboard**. In der Seitenleiste (Desktop) oder unten (Mobile) findest du:

| Tab | Wo du das findest |
|-----|-------------------|
| **Dashboard** | Deine Übersicht: nächste Klasse, XP, Belt-Fortschritt, Motivation |
| **Buchen** | Klassen buchen, eigene Buchungen ansehen |
| **Training** | Dein Trainingstagebuch, Stimmungs-Chart, Skill-Radar |
| **Turniere** | Anstehende Turniere sehen und anmelden |
| **BJJ Regeln** | Regel-Quiz zum BJJ-Wettkampfregeln lernen |
| **Leaderboard** | XP-Rangliste aller Mitglieder |
| **Gürtel** | Dein aktueller Gurt und Fortschritt |
| **Skills** | BJJ-Techniken: was du kannst, was du üben solltest |
| **Konto** | Deine Profildaten, Abo, Sprache |

## Training buchen

1. Gehe zu **Buchen**
2. Sieh dir den Wochenplan an
3. Klicke auf eine Klasse → **Buchen**
4. Bei vollen Klassen kommst du auf die Warteliste
5. Unter **Meine Buchungen** kannst du Buchungen stornieren

## Check-In

Wenn du im Gym ankommst:
- Zeig deinen **QR-Code** (Dashboard → MemberQRCode) dem Coach
- Der Coach scannt dich ein → Check-In erledigt
- Du bekommst automatisch **XP** für den Besuch

## Turniere

### Turnier-Übersicht
Gehe zu **Turniere** — du siehst alle anstehenden, vom Owner genehmigten Turniere mit:
- Name, Datum, Ort, Typ (Intern/Extern)
- Beschreibung, Anmeldeschluss
- Dein Status (falls bereits angemeldet)

### Für ein Turnier anmelden
1. Klicke **Anmelden** auf dem Turnier-Kartenrand
2. Gib an:
   - **Gewichtsklasse** (z.B. "-70kg")
   - **Gi / No-Gi / Beides**
   - **Notizen** (optional)
3. Klicke **Anmeldung senden**
4. Dein Status: **Ausstehend** — wartet auf Coach-Review
5. Nach Review wird es entweder **Bestätigt ✓** oder **Abgelehnt**
6. Bei Ablehnung kannst du dich nochmal anmelden (vor Anmeldeschluss)

## Gamification — XP & Badges

Du sammelst **XP** für:
- Training-Check-Ins
- Training-Log-Einträge
- Quiz-Erfolge
- Regel-Spiel
- Skills gemeistert

Deine Badges und dein XP-Level siehst du auf dem Dashboard. Die Leaderboard zeigt die Top-Mitglieder.

## BJJ-Regeln lernen

**BJJ Regeln** → interaktives Regel-Spiel. Frage und Antworten, lerne IBJJF-Punktestruktur spielerisch.

## Training-Tagebuch

Nach jedem Training: klick auf **Training loggen** (Banner auf Dashboard).
- Wähle die Session
- Stimmung (1–5)
- Notizen
- Technik-Tags

Das füttert deine Charts: Häufigkeit, Stimmungs-Trend, Skill-Radar.

## Gurt-Fortschritt

**Gürtel** → dein aktueller Belt und der Weg zum nächsten.
- Zeigt erforderliche Monate + Trainings
- Promotion-Historie (wann du welchen Gurt bekommen hast)

## Skills

**Skills** → BJJ-Technik-Übersicht nach Kategorie (Guard, Mount, Submissions etc.).
- Markiere Skills als "im Training" oder "gemeistert"
- Coaches können deine Skills einsehen
- Steuert deinen **Skill-Radar** im Training-Tab

## Konto

**Konto** → deine Profildaten ändern:
- Name, Telefon, Geburtsdatum
- Sprache (Deutsch / English)
- Abo-Informationen einsehen
- Passwort ändern
